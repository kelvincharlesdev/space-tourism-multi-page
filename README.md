## Sobre o projeto

Projeto para praticar conhecimentos visa a criação de uma web page responsiva, feita em React, Typescript e Styled Componente que nos conecta com um guia de turismo espacial.

## Objetivos principais

Desenvolver com React uma web page funcional conforme o desafio do [Frontend Mentor](https://www.frontendmentor.io/challenges/space-tourism-multipage-website-gRWj1URZ3/hub).

### HOME

- Header
- Menu com navegação
- Introdução do guia com botão de navegação

### Destination

- Sub Menu
- Navegação entre destinos

### Crew

- Sub Menu
- Navegação entre membros da equipe

### Thechnology

- Sub Menu
- Navegação entre tipos de tecnologias a serem usadas

## Executando o projeto localmente

> Projeto iniciado usando create vite@latest.

- Clone o projeto com

  > git clone https://github.com/kelvincharlesdev/space-tourism-multi-page.git

- Vá para a raiz do diretório

  > cd space-tourism

- Instale as dependências

  > npm install

- Inicie o servidor local

  > npm run dev

- Analisar erros padrões com ESLint

  > npm run lint

- Formata código corretamente

  > npm run format
